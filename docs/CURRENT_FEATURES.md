## 🔗 使用说明

### 快速开始

**1. 配置 API Key**
```
1. 前往设置页面 (/settings)
2. 选择 AI 提供商
3. 输入 API Key
4. 点击"测试连接"
5. 保存配置
```

**2. 创作剧本**
```
1. 前往创作页面 (/create)
2. 输入剧本主题
3. 选择风格
4. 设置集数和时长
5. 点击"开始生成"
6. 等待 AI 生成
7. 复制结果
```

**3. 批量生成**
```
1. 前往批量生成页面 (/batch-create)
2. 输入多个主题 (每行一个)
3. 设置统一参数
4. 设置并发数量
5. 点击"开始批量生成"
6. 查看实时进度
7. 导出结果
```

**4. 查看历史**
```
1. 前往历史记录页面 (/history)
2. 浏览时间轴记录
3. 点击"查看详情"
4. 点击"重新生成"
5. 导出历史记录
```
